# 玩具出口合规审查AI系统

这是一个基于百度图像识别API和沙特SASO规范的玩具出口合规智能审查系统。

## 功能特性

✓ **单个图片审查** - 快速判断单个玩具图片是否符合沙特出口要求
✓ **禁忌动物检测** - 自动识别禁忌动物（如猪、狗等）
✓ **女性形象检查** - 检查是否含有不符合伊斯兰文化的女性形象
✓ **颜色合规检查** - 验证玩具颜色是否违反沙特红黄色禁忌
✓ **宗教符号检测** - 识别可能的宗教冒犯内容
✓ **批量处理** - 支持对整个目录的图片进行批量审查
✓ **详细报告** - 生成JSON和TXT格式的完整审查报告

## 审查规则

系统检查以下关键项：

### 禁忌动物
- 猪（宗教禁忌）
- 狗（视为不洁净）

### 女性形象
- 未着罩袍的女性卡通形象
- 女性面部特写（无头巾）

### 禁忌颜色
- 纯红（RGB:255,0,0）- 血腥关联
- 亮黄（RGB:255,255,0）- 丧葬关联

### 宗教符号
- 清真寺、先知形象、古兰经文字等
- 会提示需要手工审核

## 审查报告说明

系统生成的报告包括：

1. **整体审查结果** - 合规/不合规
2. **合规评分** - 0-100分
3. **识别元素列表** - AI识别出的图片内容
4. **发现的问题** - 违规项详情和原因
5. **改进建议** - 具体的修改意见

### 合规评分含义
- 90-100分：完全合规，可以出口
- 70-89分：基本合规，建议复审
- 50-69分：存在问题，需要修改
- 低于50分：严重违规，需要重新设计

## 输出文件

运行审查后，系统会自动生成：

- `compliance_report_[图片名].json` - 结构化报告
- `compliance_report_[图片名].txt` - 文本报告

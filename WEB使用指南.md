# 🌐 玩具出口合规审查系统 - Web版使用指南

## 📋 打开网页的详细步骤

### 方法一：使用启动脚本（最简单）⭐

1. **找到项目文件夹**
   - 打开文件夹：`C:\Users\Lenovo\Desktop\task`

2. **双击启动文件**
   - 找到文件：`start_web.bat`
   - 双击运行它

3. **等待服务器启动**
   - 会弹出一个黑色的命令行窗口
   - 看到类似以下信息说明启动成功：
     ```
     玩具出口合规审查系统 - Web服务
     请在浏览器中访问: http://127.0.0.1:5000
     * Running on http://0.0.0.0:5000
     ```

4. **打开浏览器**
   - 打开任意浏览器（Chrome、Edge、Firefox等）
   - 在地址栏输入：`http://127.0.0.1:5000`
   - 按回车键

5. **开始使用！** 🎉

### 方法二：使用命令行

1. **打开PowerShell或命令提示符**
   - 按 `Win + R`
   - 输入 `cmd` 或 `powershell`
   - 按回车

2. **进入项目目录**
   ```bash
   cd C:\Users\Lenovo\Desktop\task
   ```

3. **启动服务器**
   ```bash
   python app.py
   ```
   或者使用虚拟环境：
   ```bash
   .venv\Scripts\python.exe app.py
   ```

4. **打开浏览器访问**
   - 浏览器地址栏输入：`http://127.0.0.1:5000`

## 🎯 如何使用网页界面

### 步骤1：上传图片
有两种方式上传图片：

**方式A - 点击上传**
1. 点击网页中间的紫色上传区域
2. 在弹出的文件选择窗口中选择玩具图片
3. 点击"打开"

**方式B - 拖拽上传**
1. 从文件夹中找到玩具图片
2. 直接拖拽到网页的上传区域
3. 松开鼠标

### 步骤2：预览图片
- 上传成功后，会自动显示图片预览
- 下方的"开始审查"按钮会变为可点击状态

### 步骤3：开始审查
1. 点击"开始审查"按钮
2. 等待分析（会显示旋转的加载动画）
3. 通常需要几秒钟时间

### 步骤4：查看结果
审查完成后，页面会自动显示：

✅ **总体判断**
- 显示"✅ 可以出口"或"❌ 不可出口"

📦 **检测到的对象**
- 列出识别到的所有物品
- 显示每个对象的置信度百分比

⚠️ **违规项**（如果有）
- 详细说明哪些地方不合规
- 解释违规原因

💡 **改进建议**（如果有）
- 提供修改建议

📊 **详细检查项**
- ✅ 无禁忌动物
- ✅ 无女性形象
- ✅ 无宗教符号
- ✅ 颜色合规

## ⚠️ 常见问题

### 问题1：打开浏览器显示"无法访问此网站"
**解决方法：**
- 确保启动脚本窗口没有关闭
- 检查地址是否正确：`http://127.0.0.1:5000`
- 尝试使用：`http://localhost:5000`

### 问题2：上传图片后没有反应
**解决方法：**
- 检查图片格式是否支持（PNG、JPG、JPEG、GIF、BMP）
- 确保图片文件大小不超过16MB
- 刷新页面重试

### 问题3：显示"处理失败"错误
**解决方法：**
- 检查config.py中的百度API密钥是否正确配置
- 确保网络连接正常
- 查看启动窗口中的错误信息

### 问题4：置信度显示NaN%
**已修复！** 如果仍然出现，请确保：
- 重新启动服务器
- 清除浏览器缓存并刷新页面
